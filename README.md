# 🌤 Weather Dashboard Pro

A modern weather dashboard built using React, Material UI, and WeatherAPI. Users can search for any city and get real-time weather information including temperature, humidity, wind speed, weather conditions, and weather icons.

---

## 🚀 Features

- Search weather by city
- Real-time weather updates
- Weather condition icons
- Temperature display
- Humidity information
- Wind speed information
- Loading spinner
- Error handling for invalid cities
- Default city weather on page load
- Material UI based responsive design

---

## 🛠 Tech Stack

- React
- Vite
- JavaScript
- Material UI (MUI)
- WeatherAPI

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/abhishek12-max/weather-dashboard-pro.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
